思路
--
題目其實也蠻簡單，要求用雙指針。    
先看程式
```
pin1 = 0
pin2 = len(nums)
        
while pin1 != pin2:
    if nums[pin1] == 0:
        nums.append(0)
        del nums[pin1]
        pin2 = pin2 - 1
    else:
        pin1 = pin1 + 1
                
```
設置了兩個指針，一個指向頭，一個指向尾。    

邏輯
--
但頭指針指的元素等於 0 ，則 append 一個 0 到末尾，     
隨即刪除頭指針指向的這個 0，    
最後讓末尾的指針向頭的方向走一步。    
若頭指針指的元素不等於 0 ，頭指針向末尾方向走一步。    

*邏輯就是頭指針不斷掃除 0 且增加到末尾，這時尾指針指向了新增加的 0 ，因此要走一步。*

停止條件：兩個指針相遇。
